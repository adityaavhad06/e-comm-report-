# e-comm-report-
# 🛒 E-Commerce Sales and Revenue Analytics

## 📌 Overview
This project presents a comprehensive data analytics study of an e-commerce platform using a **large-scale dataset (~1 million transactions)**.  
The goal is to analyze revenue, profit, customer behavior, and operational factors using **EDA, statistical analysis, and interactive dashboards**.

---

## 📄 Abstract
The project analyzes a diversified e-commerce dataset across five product categories:
- Electronics  
- Clothing  
- Home & Garden  
- Sports  
- Books  

It applies:
- Exploratory Data Analysis (EDA)  
- Statistical hypothesis testing (ANOVA, Chi-Square)  
- Data visualization  
- Business intelligence dashboards (Power BI)  

Key insights include revenue distribution, customer segmentation, shipping impact, and profitability trends.

---

## 🎯 Objectives
- Perform large-scale EDA on e-commerce data  
- Engineer business metrics (Revenue, Profit, Margin)

## 🔧 Methodology

### Data Preprocessing
- No missing values or duplicates  
- Data type validation  
- Clean and structured dataset  

### Feature Engineering
- Discounted Price = Price × (1 - Discount)  
- Revenue = Discounted Price + Tax  
- Profit = Discounted Price - Shipping Cost  
- Profit Margin (%)  

### EDA
- Univariate, bivariate, multivariate analysis  
- Correlation heatmaps  
- Scatter plots, histograms, boxplots  

---

## 📈 Analysis & Results
- All categories show **equal revenue (~$967/order)**  
- Average profit ≈ $854/order  
- Shipping cost is the **main expense (~$113/order)**  
- Return rate ≈ **10.49% (uniform across categories)**  
- Discount has **no strong effect on return rate**  
- Dubai is the **top revenue location**
